# AI Documentation

This repository contains an introductory guide to modern artificial intelligence (`ai.adoc`).

## Overview

The material introduces how AI models are built and how they work. It explains:

- basic neural network structure with layers, nodes, weights, and bias
- how input data is represented as vectors and processed through forward flow
- modern AI architectures built from transformer-style blocks with attention and residual connections
- training using backpropagation and gradients
- text representation with tokens and embeddings
- how chatbot-style models construct responses
- simple Python examples that connect theory with working model code

## Purpose

The goal is to provide a practical, accessible bridge between AI theory and implementation. The focus is on giving a realistic understanding of the main components of modern AI systems.

## Related files

- `ai.adoc` – the main document containing the full introduction and chapters on AI architecture, history, and model operation.
- `ai-tools.md` – a companion file listing common AI tools of the day.
